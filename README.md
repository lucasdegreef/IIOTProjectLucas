# BigProject

This project contains manufacturing dataset analysis and Codesys project files for Git integration.

## Project Structure
- `dataset/`: Contains the manufacturing dataset and analysis notebook.
- `CodesysForgit/`: Contains the Codesys project files optimized for Git.

## Getting Started
To use this project, clone the repository and explore the contents of the `dataset/` and `CodesysForgit/` directories.
